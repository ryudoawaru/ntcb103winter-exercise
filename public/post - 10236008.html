關於Ruby是甚麼呢?

Ruby是由一位日本人松本行弘開發的，屬於物件導向的語言。特點為：免費、open source、容易學，所以現在很多人都在學Ruby，我也不例外，哈哈哈XDDD
在Ruby中所有你看到的都是物件(Object) 物件         物件                  物件 很重要！所以要講三次！
Ruby是電腦程式語言，而Ruby on Rails(RoR)是一種使用Ruby建構出來的網頁開發框架，RoR不是程式語言喔！

首先來安裝Ruby吧！

  1. 我們到Ruby中文官網 https://www.ruby-lang.org/zh_tw/ 下載安裝，也可以直接點選這個 http://rubyinstaller.org/ 下載安裝
  2. 再來安裝好用的文字編輯器Subline 。 http://www.sublimetext.com/
  3. 接著就可以開始Hello Ruby囉！

ps.如果電腦上沒有Ruby的執行環境，不用擔心，可以利用 nitrous 來執行Ruby，只需要註冊就可以了^_^

我們可以檢查看看Ruby是否有安裝成功，開啟命令提示字元cmd，輸入ruby -v，就可以看到現行安裝的版本： 
ruby 2.1.5p273 (2014-11-13 revision 48405) [i386-mingw32]

Gem簡介

RubyGems是Ruby的套件管理系統，讓你輕易安裝及管理Ruby函式庫。你可以在RubyGems上找到所有的Ruby開源套件。

常用的指令

gem -v  告訴你 RubyGems 的版本
gem install PACKAGE_NAME  安裝某個套件
gem env  RubyGems細項
gem list  列出安裝的套件

開始來學Ruby基礎囉！

變數、常數、保留字&關鍵字

變數：

變數 意思
區域：var 通常存活在某個區域中，可能是 def ~ end 的範圍內，不同區域可以有相同名稱
全域：$var 這就非常厲害了，在程式的任何地方都可以使用，只要名稱相同，就視為同一個變數PS. 之前看過某位大大的見解：全域變數前面加的符號是 "$" 字號，所謂「有錢能使鬼推磨、有錢就是大爺」，所以這個 $ 字號的變數勢力遍及整個程式，真是非常有趣的解釋呀^~^!!
實體：@var 就是在實體化的類別中存在的變數，不同的實體，有獨自的變數
類別：@@var 不管有幾個實體物件，只要屬於同一個類別所實體化出來的，通通共用一個變數

常數：大寫字母開頭，而且內容可以被改變

保留字&關鍵字：就是變數名不能跟它取的一樣啦

5.upto,downto

1.upto(10) do |i|
    puts "hi, ruby #{i}"    #1到10，升冪
end
10.downto(1) do |i|
    puts "hi, ruby #{i}"    #10到1，降冪
end

names.each_with_index do |name,x|

end

方法(Method)

  1. 小括號可以省略

  2. ? !是程式的一部分

  3. 回傳值不用寫return

def is_adult?(age)
    if age >=18
       true #(不用寫return)
    else
       false
    end
end
p is_adult?(2)
#與下列程式碼相同
def is_adult?(age)
    age >=18
end
p is_adult?(20)
執行結果： 
false
true
[Finished in 0.2s]

物件導向

在Ruby上是很重要的觀念，因為Ruby所有的東西都是物件
以下是一個小小的範例： 
class Animal #定義Animal類別會吃會睡
  def eat
    puts "yummy!"
  end
  def sleep
    puts "sleep"
  end
end
class Cat < Animal #cat類別繼承Animal
  def play #cat專屬會play
    puts "Cat play fish"
  end
end
class Dog < Animal #Dog類別也繼承Animal
  def run #Dog專屬會run
    puts "Dog run run run"
  end
end
d=Dog.new #產生一個新物件
c=Cat.new
c.eat #實作Animal類別的eat
c.play #實作Cat類別的play
d.run #實作Dog類別的run
d.eat #實作Animal類別的eat
執行結果： 
yummy!
Cat play fish
Dog run run run
yummy!
[Finished in 0.2s]

存取控制

跟其他的程式語言一樣，有這些方法。 在Ruby裡 99.9%的protected都是在寫private
public：屬性和方法可以在程式中任何地方直接存取 protected：屬性和方法可以在相同文件或是子類別中存取 private：屬性和方法只在在類別中存取 在這裡有詳細說明

模組(Module)

  1. 無法繼承

  2. 無法實體化

module Flyable #建立Flyable模組
  def fly #定義fly方法
    puts "I can fly"
  end
end
class Animal #建立Animal類別
end
class Bird < Animal #建立Bird類別，繼承Animal
  include Flyable #include Flyable模組
end
bird = Bird.new
bird.fly
執行結果： 
I can fly
[Finished in 0.2s]
